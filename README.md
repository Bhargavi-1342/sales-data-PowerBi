# Sales Data Analysis – Power BI Dashboard

This project presents a Power BI dashboard that analyzes sales performance and business trends. The dashboard helps visualize important sales metrics such as total revenue, product performance, and monthly sales trends.

The goal of this project is to explore how factors like product category, region, and time period influence overall sales performance.

## Key Insights

* Certain product categories generate higher sales revenue.
* Monthly trends reveal peak sales periods.
* Some regions contribute more to total revenue than others.
* High-demand products drive overall business growth.
* Sales performance varies across different time periods.

## Dashboard Features

* Interactive filters for product category and region
* Visualization of sales by product and category
* Monthly sales trend analysis
* Revenue comparison across regions
* Summary indicators showing total sales and total profit

## Tools and Technologies

* Power BI
* Data Visualization
* Data Analysis

## How to Use

1. Download the `.pbix` file from the repository.
2. Open the file using Power BI Desktop.
3. Explore the dashboard visuals and use filters to analyze sales insights.
